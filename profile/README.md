# SkelForm

SkelForm is a free & open-source 2D skeletal animator, designed for easy integration into games.

## Editor

The [official editor](https://github.com/Retropaint/SkelForm) requires [Rust](https://rust-lang.org/) to be compiled, but is otherwise straightforward and can be compiled on Windows/Mac/Linux/Web.

Forks and/or 3rd-party editors are welcome, though no documentation is provided at the moment and the above editor is the only reference.

## Official Runtimes

Check out the [runtime tables](https://skelform.org/#runtimes) for a list of runtimes created and maintained by SkelForm's developer ([Retropaint](https://github.com/retropaint)]. These are often the first to be updated and can be used as references for developing your own runtimes.

## Community Runtimes

Below is a (not exhaustive) list of runtimes maintained by the community. 

- [Godot runtime](https://github.com/MudkipWorld/SkelFormGD)
- [C Runtime](https://github.com/BatteRaquette581/skelform_c)

*Note: these are not guaranteed to be up-to-date. You can help them out by sending issues/PRs in their respective repos.*

## Miscellaneous

- [GD-SWF](https://github.com/MudkipWorld/GD-SWF) - A general Shockwave Flash (SWF) converter with support for SkelForm
